# Test Plan

## Project Information

**Project Name:** ECommerce QA Automation Framework

**Application Under Test (AUT):** https://automationexercise.com

**Project Type:** Manual Testing and Test Automation Portfolio Project

**Objective:**
To validate the functionality, usability and reliability of the Automation Exercise e-commerce platform through structured manual testing and automated test execution.

---

## Scope

### In Scope

* User Registration
* User Login / Logout
* Product Search
* Product Details
* Cart Management
* Checkout Process
* Contact Us Form
* Newsletter Subscription
* Account Deletion

### Out of Scope

* Payment Gateway Validation
* Browser Performance Testing
* Security/Penetration Testing
* Mobile Application Testing

---

## Testing Types

* Functional Testing
* Smoke Testing
* Regression Testing
* Integration Testing
* Exploratory Testing
* User Acceptance Testing (UAT)

---

## Test Environment

**Operating System:** Windows 11

**Browser:** Google Chrome

**Automation Tool:** Selenium with Python

**Test Framework:** Pytest

**Version Control:** Git & GitHub

---

## Deliverables

* Test Plan
* Risk Matrix
* Test Scenarios
* Defect Reports
* Selenium Automation Framework
* Test Execution Reports
* Screenshots

---

## Entry Criteria

* Application is accessible
* Test environment is available
* Test data is prepared

---

## Exit Criteria

* Planned test scenarios executed
* Critical defects documented
* Test execution report generated
* Automation scripts successfully executed
